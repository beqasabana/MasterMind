Welcome to MasterMind

Game where player tries to brake a code made by computer or another player

Rules:

1) Codes include numbers from 0 to 7.
2) Duplicate numbers are allowed.
3) Player has 10 tries to brake the code.

UI:

Red circle indicates correct number in correct position.
Yellow circle indicates correct number in wrong position.
Blank(white) circle indicates wrong number.

note: order of circles is not representation of code order
